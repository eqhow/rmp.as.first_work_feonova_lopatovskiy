Работа №1. Создание игрального кубика. Работу выполнили Лопатовский Родион и Феонова Елена. ИР-23-4

При нажатии кнопки "Бросить кубик" отображается рандомная картинка грани кубика с числами от 1-6.
Для кнопки использовали "Button", для изображения - "ImageView"
